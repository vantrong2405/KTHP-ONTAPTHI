# iKaraoke
Bạn thích hát karaoke? Ứng dụng iKaraoke sẽ giúp bạn điều ấy. Ứng dụng giúp bạn hát karaoke ngay trên điện thoại di động của mình. iKaraoke với kho bài hát rộng lớn và luôn luôn cập nhật.
